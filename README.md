# 🚀 MERN TODO App

A full-stack task management application built using the MERN stack (MongoDB, Express, React, Node.js).

This project demonstrates modern full-stack development practices with a focus on clean architecture, user experience, and responsive design.

---

## 🧠 Overview

This application allows users to:

- Create, update, delete tasks
- Mark tasks as completed
- Filter, search, and sort tasks
- Manage priorities and due dates
- Experience a modern UI with dark mode and animations

The app is designed to go beyond a basic TODO list by including real-world usability features.

---

## ✨ Key Features

- Full CRUD operations
- Mark tasks as completed
- Filtering (All / Active / Completed / Priority)
- Real-time search
- Sorting options
- Priority levels (Low / Medium / High)
- Due date management
- Optimistic UI updates
- Toast notifications
- Dark mode support
- Floating Action Button (FAB)
- Progress tracking
- Persistent UI state (localStorage)

---

## 🛠 Tech Stack

### Frontend
- React (Vite)

- Axios

### Backend
- Node.js
- Express.js

# 🎨 TODO App Frontend

## 📝 Overview

This is the frontend of a full-stack MERN TODO application built using React and Tailwind CSS.

The application provides a modern, responsive UI for managing tasks with advanced features like filtering, search, sorting, priority levels, due dates, and dark mode.

---

## ✨ Features

- View all todos
- Create new todos (via Floating Action Button + modal)
- Edit todos (inline editing)
- Delete todos with confirmation
- Mark todos as completed / not completed
- Filter tasks:
  - All
  - Active
  - Completed
- Filter by priority (Low / Medium / High)
- Real-time search by title
- Sorting (Newest / Oldest / Completed)
- Progress bar (task completion)
- Toast notifications (success & error)
- Dark mode toggle
- Responsive UI (mobile-first)
- Persistent UI state (theme, filters, search, sort)

---

## 🛠 Tech Stack

- React (Hooks-based)
- Vite
- Tailwind CSS
- Axios

---

## ⚙️ Setup Instructions

### 1. Navigate to frontend directory

```bash
cd client
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment (optional)

```bash
VITE_API_URL=http://localhost:5000/api/todos
```

### 4. Run the application

```bash
npm run dev
```
### App will run on:
```bash
http://localhost:5173
```
